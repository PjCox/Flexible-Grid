Flexible Grid System

Responsive

Built for responsive sites, Flexible Grid allows you to build layouts on a foundation suited for mobile and tablet browsers.

Small Footprint

Flexible Grid is simply flexible, a grid. Flexible Grid is not a framework, it's a CSS column system for quickly laying out your elements.

Column Layout

Built with a twelve column structure of Flexible Grids, dividing easily into columns of upto 12 in width, giving developers many layout possibilities.
